Certificate

Linux Based Installation:

Run plgx_cpt to view the help options.

Follow these steps to grant the Full Disk Access privilege to the EclecticIQ Endpoint Response client:
  1. In the Security & Privacy dialog box, switch to the Privacy tab and select Full Disk Access.
  2. Click the padlock icon to edit the settings.
  3. Expand the Full Disk Access category by clicking the + button.
  4. Press CMD + Shift + G, enter the name of client file (/user/local/bin/plgx_agent).
  5. Select the plgx_agent file and click Open.
  6. Scroll and verify the client file is added to the list.
  7. Repeat steps 4 through 7 for the /usr/local/bin/plgx_osqueryd and /usr/libexec/atrun files.
